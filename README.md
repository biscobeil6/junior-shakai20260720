# 社会暗記アプリ v0.3（周回版）

## 主な変更
- 歴史・地理・公民の全496問を収録
- 分野選択 → 単元選択 → モード選択の流れ
- 歴史には「全時代」を追加
- 周回モード：一周するまで同じ問題を出さない
- 間違い見直し：直近の回答が不正解の問題だけ出題
- 復習期限・間隔反復を廃止
- 途中で閉じても周回状況を端末内に保存
- 正解選択肢に○、誤答選択肢に×を表示
- 成績リセットで回答履歴・間違い・周回状況を削除
- app.js / style.css に v=3 を付けてキャッシュ対策
